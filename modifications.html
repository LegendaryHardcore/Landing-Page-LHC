SERVER MODIFICATIONS

The server is mostly vanilla. There are no land claims, no economy plugins and no factions plugins, but we have made some additions that make the server unique:

  • When a player dies their head is placed at the location of their death
  • By holding a dead player’s head and running the command /deadenderchest, a living player can access the contents of a dead player’s ender chest. This can only be executed once.
  • Phantoms can be toggled once a player has got the ‘two birds, one arrow’ advancement by running /phantoms
  • Withers and lava casting is disabled in the spawn region between -3000, -3000 and 3000, 3000
  • When a player enters the end they are spawned at a random location on the main end island rather than the end platform.
  • An obsidian block can be ‘tamed’ by looking at it and running /petobby. A ‘pet obby’ will transform in to crying obsidian when the player is offline, returning to obsidian when the player returns. This can be used to create ender pearl stasis chambers.
  • Each player spawns with a journal that can only be edited by them and is automatically signed on death. The journal is indestructible and will never despawn.
